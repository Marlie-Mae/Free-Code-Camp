Introduction

C# is a modern, object-oriented programming language developed by Microsoft. It is part of the .NET framework and is designed for building a variety of applications, including web, mobile, and desktop applications.

C# provides a rich set of features, including strong type checking, garbage collection, and support for both imperative and functional programming paradigms. Its syntax is similar to that of other C-based languages, making it relatively easy to learn for developers familiar with C, C++, or Java.

  • C# extends the core language by providing a robust set of libraries and frameworks that enhance application development. For example, C# allows developers to create applications that interact with the .NET framework, manage data through Entity Framework, and utilize various APIs for web and desktop applications.
  • C# also supports server-side programming, allowing developers to build applications that can communicate with databases, manage user sessions, and handle file operations. Frameworks like ASP.NET enable developers to create dynamic web applications that can process user requests and serve responses efficiently.
What you should already know

This guide assumes you have the following basic background:

  • A general understanding of programming concepts and paradigms.
  • Familiarity with object-oriented programming principles.
  • Some experience with a programming language, preferably C or Java.
C# and Java

While C# and Java share similarities in syntax and design principles, they differ in many aspects. C# was developed specifically for the Windows platform and integrates seamlessly with the .NET framework, while Java is platform-independent and runs on the Java Virtual Machine (JVM).

C# supports advanced features like properties, events, and delegates, which do not exist in Java in the same form. Additionally, C# has features like LINQ (Language Integrated Query) and async/await for asynchronous programming, enhancing its capabilities.

Hello world
To get started with writing C#, you can create a simple console application and write your first "Hello world" C# code: using System; class Program { static void Main() { Console.WriteLine("Hello, World!"); } } Run this code in your C# environment to see the output!
Variables

In C#, variables are used to store data that can be changed during program execution. Variable names must follow specific rules, starting with a letter or underscore, followed by letters, numbers, or underscores.

C# is case-sensitive, meaning that "variable" and "Variable" would be considered two different identifiers.

Declaring variables
You can declare a variable in several ways:

Using the var keyword for type inference. For example, var x = 42; automatically infers that x is of type int.

Explicitly specifying the type. For example, int y = 13; declares y as an integer.

Variable scope

In C#, variables declared within a method are local to that method, while those declared outside are accessible throughout the class. Understanding scope is essential to avoid unintended behavior in your programs.

Variables can be declared with different access modifiers (public, private, protected) to control their visibility.

Global variables

In C#, global variables can be declared using static fields within a class. These variables are accessible from any method in that class but should be used sparingly to maintain code clarity.

Constants

You can create a read-only constant with the const keyword in C#. The identifier must start with a letter or underscore and can contain letters, numbers, and underscores.

const double PI = 3.14;

A constant must be initialized at the time of declaration and cannot be changed afterward.

Data types

The C# language defines several data types:

  • Primitive data types:

    • int for integers.
    • double for floating-point numbers.
    • bool for true/false values.
    • string for text.
  • Reference types, such as classes and arrays.
C# also supports nullable types, allowing variables to represent undefined values.
if...else statement
Use the if statement to execute a block of code if a condition is true. You can also use else to execute a block if the condition is false. An if statement looks like this: if (condition) { /* statements */ } else { /* statements */ }

C# allows for nested if statements and you can chain conditions using else if:

if (condition1) { /* statements */ } else if (condition2) { /* statements */ } else { /* statements */ }
while statement
A while statement repeatedly executes its statements as long as the specified condition is true. It looks like this: while (condition) { /* statements */ } If the condition becomes false, the loop terminates.

For example, the following while loop increments a variable until it reaches three:

int n = 0; while (n < 3) { n++; }
Function declarations
A function in C# is defined using the method keyword, followed by the return type, name, and parameters:
  • The return type of the function.
  • The name of the function.
  • Optional parameters enclosed in parentheses.
For example: public int Add(int a, int b) { return a + b; }
